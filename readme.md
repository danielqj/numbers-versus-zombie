# Numbers Versus Zombie

一个给孩子练习一位数乘法口诀的浏览器小游戏。

## 玩法

- 僵尸从屏幕左边进入，向右边的堡垒移动。
- 每个僵尸头上都有一道一位数乘以一位数的题目。
- 玩家在答案框输入正确答案后，离堡垒最近的僵尸会爆炸并得分。
- 输入错误时，当前僵尸会立刻向堡垒前进一段距离。
- 连续答对 5 题会获得 5 分 bonus。
- 分数到达 100 分前会进入 Boss 关。
- Boss 必须连续答对 5 题才能消灭。
- 消灭 Boss 后宣布人类获胜并播放胜利特效。

## 难度

下方的难度滑杆可以调整游戏速度。难度越高，僵尸移动越快，出现也越频繁。

## 文件

- `index.html`：页面结构
- `styles.css`：界面和响应式样式
- `game.js`：游戏逻辑、动画、判题、Boss 和胜利特效
- `readme.md`：项目说明

## 运行

直接用浏览器打开 `index.html` 即可游玩，不需要安装依赖。
